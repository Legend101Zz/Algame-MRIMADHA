Welcome to Algame Documentation
=============================

.. image:: _static/logo.png
   :align: center
   :alt: Algame Logo

**Algame** is a modern algorithmic trading and backtesting framework designed to make algo trading accessible without compromising power.

Key Features
-----------

* Multi-Asset Trading Support
* Visual Strategy Building
* Advanced Risk Management
* Real-time Backtesting
* Comprehensive Analysis Tools
